# External Service Registrations

This directory contains External Service Registration metadata files that define OpenAPI schemas and operations for MCP services.

## Included Services

- AccountAnalytics - Account data aggregations
- OpportunityMcpService - High-value opportunity queries
- OpportunityMcpServiceAuraEnbaled - AuraEnabled version
- PromptTemplateMcpService - Prompt template execution
- GetCaseByCaseNumber - Case lookup by number
- GetOpportunitiesByAmount - Opportunity filtering
- InvokeAgentAction - Agent invocation
- MyCustomAPI - Custom REST API

Each registration defines the OpenAPI schema and authentication requirements for the corresponding service.
